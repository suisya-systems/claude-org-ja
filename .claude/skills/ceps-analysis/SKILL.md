# ceps-analysis: CEPs競合分析レポート生成

ExcelベースのCEPsアンケートデータから、ブランドの競合ポジショニングを統計的に分析し、
マーケターフレンドリーなアクション提案付きレポートを生成する。

## 背景

マンナンライフ蒟蒻畑シリーズのCEPs分析で初めて使われたパターン。
5ブランド×1000サンプルのアンケートデータから、利用シーンと体験価値の
メンタルアドバンテージを統計的に算出し、マーケティング施策の優先順位付けを行った。

## 前提条件

- Python 3.8+ がインストールされていること
- `openpyxl` ライブラリが利用可能であること（`pip install openpyxl`）
- 入力データ: Excel形式（.xlsx）のCEPsアンケートデータ
  - 複数シート構成（ブランド別 or シーン別）
  - 利用シーンと体験価値のクロス集計が可能なデータ構造

## 手順

### Step 1: データ構造の解析

Excelファイルを開き、シート構成とデータレイアウトを把握する。

```python
import openpyxl

wb = openpyxl.load_workbook('input.xlsx', data_only=True)
for sheet_name in wb.sheetnames:
    ws = wb[sheet_name]
    print(f"Sheet: {sheet_name}, Rows: {ws.max_row}, Cols: {ws.max_column}")
    # 先頭5行を出力してヘッダー構造を確認
    for row in ws.iter_rows(min_row=1, max_row=5, values_only=True):
        print(row)
```

**重要**: pandas ではなく openpyxl を直接使う。理由:
- 複雑なセル結合やマルチヘッダーに対応しやすい
- メモリ効率が良い
- 依存ライブラリが少ない

### Step 2: メンタルアドバンテージ分析エンジンの実装

調整済み残差（Adjusted Residual）を用いて、各ブランドの各CEPにおける統計的優位性を算出する。

```python
import math

def adjusted_residual(observed, row_total, col_total, grand_total):
    """調整済み残差を計算する"""
    expected = (row_total * col_total) / grand_total
    if expected == 0:
        return 0.0
    residual = observed - expected
    # 標準誤差
    se = math.sqrt(expected * (1 - row_total/grand_total) * (1 - col_total/grand_total))
    if se == 0:
        return 0.0
    return residual / se
```

**判断基準**:
- |調整済み残差| > 1.96 → 統計的に有意（5%水準）
- |調整済み残差| > 3.0 → 強い有意性

### Step 3: 競合ポジショニング分析

各CEP（利用シーン・体験価値）について、自社と競合のスコアを比較し分類する。

```python
def classify_cep(own_score, competitor_scores):
    """CEPを強み/弱み/中立に分類"""
    top_competitor = max(competitor_scores.values())
    
    if own_score > 1.96:
        if top_competitor > 1.96:
            return "防衛"  # 自社も競合も強い
        return "強化"      # 自社が強い
    elif own_score < -1.96:
        return "攻略"      # 自社が弱い（競合が強い可能性）
    else:
        return "中立"
```

### Step 4: クロス分析（黄金パターン発見）

利用シーン × 体験価値のクロス集計から、共起率の高い組合せを発見する。

- 自社の共起率が全体平均より高い → 強みの訴求ポイント
- 自社の共起率が全体平均より低い → 新規開拓の機会

### Step 5: マーケティングアクション提案の生成

分析結果を3カテゴリに分けて、具体的なアクション提案を生成する。

1. **強化すべきCEP**: 自社が統計的に有意に強い領域 → さらなる訴求強化
2. **攻略すべきCEP**: 競合が強いが市場が大きい領域 → 新規参入のチャンス
3. **防衛すべきCEP**: 自社が強いが競合も追随している領域 → ポジション維持

**提案のフォーマット**:
- 根拠: 数値データに基づく客観的説明
- 推奨アクション: マーケターが実行可能な具体的施策

### Step 6: レポート生成

Markdown形式でレポートを生成する。以下の構成:

1. 分析概要（対象ブランド・サンプル数）
2. エグゼクティブサマリー（強み/弱みの件数）
3. メンタルアドバンテージ分析（利用シーン）
   - ASCIIアートによるポジショニングバー
4. メンタルアドバンテージ分析（体験価値）
5. CEPシェア分析
6. クロス分析（黄金パターン）
7. ポジショニングマップ（ASCIIアート）
8. マーケティングアクション提案
9. 付録（分析手法の説明）

**ASCIIポジショニングバーの例**:
```
  ブランドA ──────────│█████████ +19.46 ← 自社
  ブランドB ────────██│───────── -5.69
```

## 判断基準・閾値

| 基準 | 値 | 根拠 |
|---|---|---|
| 統計的有意水準 | ±1.96 | 5%有意水準（z検定） |
| 強い有意性 | ±3.0 | 慣例的な強い有意性の閾値 |
| CEP分類: 強化 | 自社 > +1.96, 競合トップ < +1.96 | 自社のみが有意に強い |
| CEP分類: 攻略 | 自社 < -1.96 | 自社が有意に弱い |
| CEP分類: 防衛 | 自社 > +1.96, 競合トップ > +1.96 | 両者とも有意に強い |

## 応用・バリエーション

- **異なるカテゴリ**: 食品以外（化粧品、家電等）でも同じ分析フレームワークが適用可能。CEPの定義（利用シーン・体験価値）をカテゴリに合わせて変更する
- **時系列比較**: 同一ブランドの複数時点のデータがあれば、CEPシェアの推移を追加分析できる
- **セグメント別分析**: 年代・性別等のデモグラフィックでセグメント分割し、ターゲット別のCEPマップを作成できる

## 注意点

- openpyxl のみで十分。pandas は不要（複雑なセル結合への対応が逆に難しくなる）
- Excelのシート構成はプロジェクトごとに異なるため、Step 1 のデータ構造解析を丁寧に行うこと
- 統計的有意性だけでなく、市場規模（言及数）も考慮してアクション優先順位を決めること
- レポートはマーケター向けに書く。統計用語には必ず平易な説明を付ける
- ASCIIアートのポジショニングバーは視認性が重要。幅を揃え、自社には `← 自社` マーカーを付ける
