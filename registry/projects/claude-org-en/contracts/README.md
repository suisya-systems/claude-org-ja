# contracts/ — 参照のみ（実体を置かない）

このディレクトリには**契約の実体を置かない**。人間承認済みの独立した契約文書への
**リンクと 1 行説明の索引**だけを置く。置いてよいファイルは本 `README.md` の 1 枚のみ。

## なぜ実体を置かないか

常設スコープ契約は人間の承認を伴う独立文書である。その本文を台帳へコピーすると、
**セッション限りの特例が恒久的なプロジェクト方針へ化ける**。実際、本台帳の種データ元である
`.state/conveyor/scope-contract.md` には人間によるマージ事前承認の override が含まれ、
`approved_at` は「2026-07-18（JST セッション）」、`max_iterations` / `time_budget` は
「本セッション」、include 述語は既にクローズ済みの特定 PR / Issue 番号の列挙である。
これは**期限切れのセッション成果物**であって、このプロジェクトの契約ではない。

新しい常設スコープ契約が要るなら、**独立文書として人間承認を取り直すこと。**
台帳へのコピーはその代替にならない。

実行系（`tools/project_dossier.py`）はこのディレクトリを一切読まない。
`README.md` 以外のファイルが在ると、プロファイル解決時に警告が出る。
